Types of Regression Models
Model Algorithm
OLS (Ordinary Least Squares) OLS is a type of linear regression that estimates the parameters of a linear model. It minimizes the sum of squared residuals, ensuring the best possible fit between the observed data and the model's predictions. OLS is widely used for its simplicity and efficiency in cases where the independent variables are not highly correlated.
WLS (Weighted Least Squares) WLS is an extension of OLS that assigns different weights to data points based on their variance. This method is particularly useful when dealing with heteroscedasticity. WLS aims to provide more accurate parameter estimates by giving less weight to data points with higher variance.
GLS (Generalized Least Squares) GLS is a flexible approach to linear regression that allows for the modeling of correlations and non-constant variability in the data. GLS can handle situations where assumptions of constant variance and no autocorrelation are violated, by transforming the original model into one that can be estimated using OLS techniques.

OLS - Ordinary Least Squares

Ordinary Least Squares (OLS) is a fundamental method for linear regression analysis, aiming to find the best-fitting line through the data points by minimizing the sum of the squares of the vertical distances of the points from the line. This method assumes that there is a linear relationship between the independent variables and the dependent variable. OLS is particularly effective in scenarios where the data does not show significant multicollinearity and the error terms are homoscedastic and independently distributed.
